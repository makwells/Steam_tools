# __SPC__
> __SPC__ - Steam Profile Checker, a tool that will help you get complete information about Steam profile, view the account cost, and view the cost of items in inventory, and much more.

# __Main functions of the program__
  - Check Steam Profile information 
  - Convenient price analytics for items 
  - Convenient price analytics for games and software
  - Shows the cost of inventory and profile
