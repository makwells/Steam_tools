# What is it? 
> __SPC__ - Steam Profile Checker, a tool that will help you get complete information about Steam profile, view the account cost, and view the cost of items in inventory, and much more.

## How to use?

### Profile
``` sh
#open profile menu
profile

#My profile
profile -m
```

### Wishlist store
```sh
#open wishlist menu
wishlist

#Add new game/program in wishlist
wishlist -add "id"

#Remove game/program in wishlist
wishlist -remove "id"
```
### Wishlist market 

```sh 
#Open market menu
market

#Add new item in market wishlist
market -add "name"

#Remove item in market wishlist 
market -remove "name"
```
